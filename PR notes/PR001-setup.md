# PR 1: Project Initialization & Dependencies

## Goal

Set up the Rust project structure and configure dependencies.

This PR creates the basic foundation for our CRDT library. Initialize a Rust library project, set up the configuration file, and add the dependencies needed later.

## What to Build

- A Rust library project structure
- Configuration file (`Cargo.toml`) with project info and dependencies
- Basic library file (`src/lib.rs`)
- Updated `.gitignore` for Rust build artifacts

## Reading List

- https://doc.rust-lang.org/book/ch01-00-getting-started.html
- Learn how Rust projects are structured, what `Cargo.toml` does, and basic Rust tooling

**The Rust Book - Chapter 7: Managing Growing Projects with Packages, Crates, and Modules**
- https://doc.rust-lang.org/book/ch07-00-managing-growing-projects-with-packages-crates-and-modules.html
- Understand Rust's module system and project organization

### Dependency Management

**Cargo Book - Dependency Specification**
- https://doc.rust-lang.org/cargo/reference/specifying-dependencies.html
- Learn how to specify dependencies and their versions

### Testing Basics
